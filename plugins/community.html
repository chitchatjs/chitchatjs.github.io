# 🌱 Chitchat Community

Community supported ecosystem for Chitchatjs.

# Plugins

  • @chitchatjs/alexa - 🔥 Core Alexa building blocks library.
  • @chitchatjs/plugin-ax-common - 👋 A library of some common building blocks such as fallback, stop, help handlers.
  • @chitchatjs/plugin-ax-session - 👋 A library of session management building blocks that allow you to manipulate and customize session.
  • @chitchatjs/plugin-ax-display - 👋 A library of display related building blocks, prebuilt APL layouts and more.
